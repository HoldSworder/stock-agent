# ETF 中线主升浪交易策略蓝本

> 本文档是 2026-06 一轮系统性回测与讨论的结论沉淀，定位为**策略逻辑蓝本**（纯逻辑，不含代码实现）。
> 所有结论均经过 2025/2026 跨年份、主线/非主线跨板块的真实数据回测验证。

---

## 0. 核心理念（先想清楚系统到底解决什么）

这套系统**不追求三件做不到的事**：

- ❌ 不预测顶底（顶底只能事后确认，任何"推测顶底"本质都是概率，强行精确抄顶摸底会掉回"用震荡思维预判回调"的老毛病）
- ❌ 不追求机械跑赢"买入持有"（确认的主升浪里，裸持主线就是理论最优解，机械择时的天花板是逼近它）
- ❌ 不做全自动交易（系统输出可直接照做的结构化指令，用户手动下单）

它要解决的是**执行纪律和结果闭环**这个真正的瓶颈：

| 交易痛点 | 系统如何治 |
|---|---|
| 踏空主升 | 回补层逼你在新高/趋势确认时上车 |
| 怕回调提前卖飞 | 出场只认趋势确认信号，小回调不动 |
| 顶部跑不掉 | 预警层提前收紧防线 |
| 锚定 / 凭感觉 | 全部变成明确的标记和指令 |

> **一句话：把"试仓→趋势确认加仓→趋势破位离场"的纪律，变成你能照着做的明确指令。真正的超额收益来自「选对正在走趋势的板块」，策略本身只负责执行。**

## 1. 分层架构总览

```
┌─ 选股/准入层 ── RS 相对强度 + 板块宽度          → 决定「碰哪个板块」（慢、要确定性）
├─ 入场层 ────── 日线 MACD 金叉                  → 决定「何时买」（及时）
├─ 出场层 ────── Supertrend(10,3) 翻空           → 决定「何时卖」（咬住整波、不被洗）
├─ 预警层 ────── 跌停潮 / 顶背离 / 资金背离        → 收紧止损、减仓（不自动清仓）
└─ 回补层 ────── 价格新高 + 站上MA60 + 宽度推力    → 退潮卖出后何时买回（治踏空）
```

**信号分工原则：快信号管出场，慢信号管准入。**

| 任务 | 用什么 | 速度 | 理由 |
|---|---|---|---|
| 退潮当天报警 | 跌停潮 / 涨停→跌停翻转 | ⚡ 当天 | 离散、无歧义 |
| 快速清仓 | Supertrend 翻空 | 几天内 | 不抖动、咬整波 |
| 判真假退潮 / 该不该回补 | RS 相对强度 | 🐢 慢但准 | 要确定性，不要及时性 |

## 2. 入场层：日线 MACD 金叉（不加任何过滤）

**规则：日线 MACD 金叉（DIF 上穿 DEA）即入场信号。**

### 为什么不加 ADX / 放量过滤（回测证据）

联网调研建议的 ADX>25、放量确认，实测**越加越差**：

| 标的(2026) | 买入持有 | 纯金叉 | 金叉+ADX>25 |
|---|---|---|---|
| 半导体 | +108% | +49% | **+23%↓** |
| 新能源 | +47% | +17% | +6%↓ |
| 通信 | +79% | +17% | -3%↓ |

- **ADX 是滞后指标**：趋势启动初期 ADX 还在低位（金叉时常 20~22），等 ADX 涨过 25 趋势已走一半，过滤恰恰删掉了最好的早期入场点。
- **"金叉当天放量"是错误口径**：金叉是回调后的缩量拐点，当天几乎都缩量（量比 <1.5x），该条件会误杀几乎所有真信号。放量确认只在"金叉后突破新高那天"才有意义。

> **结论：日线金叉已是天花板级入场，别再找"更准的买入信号"。MACD金叉、双均线、唐奇安突破、Supertrend翻多本质同类，差异不大；真正的差距从来不在入场那一下。**

## 3. 出场层：Supertrend(10,3) 翻空（不加深死叉）

**规则：Supertrend(ATR周期10, 倍数3) 由多翻空（线由绿转红）即清仓。**

### 为什么是 ST 而非死叉（回测证据）

| 出场口径(2026均值) | 收益 | 交易次数 |
|---|---|---|
| 金叉+普通死叉 | +28% | 8~12次（churn严重） |
| 金叉+深死叉(DIF<0) | +15% | — |
| **金叉+ST翻空** | **+47%** | **1~3次** |

- ST 的 ATR 通道在强趋势里**不抖动**，半导体/科创 全程 1 次交易咬住整波。

### 三个必须遵守的"不要"（全是踩过的坑）

1. **不要加"深死叉(DIF<0)"做出场**：MACD 金叉常发生在 0 轴下方，金叉进场时 DIF 还是负的，"DIF<0"会在下一根**立刻把仓位秒平**。深死叉只配"价格新高入场"，不配金叉入场。
2. **不要放宽到 ST(14,3)**：实测半导体 +79%→+47%、机器人 +39%→+28%。Supertrend 参数**极度敏感**，(10,3) 是甜点，换参数就崩。
3. **不要靠"加快回补"补多腿趋势的捕获缺口**：ST翻多快速回补实测与基准完全一致——丢捕获的根源是"深调时已被弹出、在更高价买回"的往返差价，任何回补加速都补不回来。

## 4. 选股/准入层：RS 相对强度 + 板块宽度（真正的胜负手）

策略只是趋势收割机，**对准趋势板块就赢，对准震荡板块就趴着**。所以"把仓位放到正在走趋势的板块"才是超额收益来源。

### 4.1 RS 相对强度（板块/大盘）——最干净的"板块准入 / 真假退潮"判据

- **定义**：RS = 板块价 ÷ 大盘价（沪深300）。
- **准入**：只碰"RS 仍创新高 / 守住领先"的板块。RS 跌破并持续走低的板块 = 退潮死板块，直接不进候选池。
- **被动 vs 主动退潮**：
  ```
  被动承压(可扛) = 板块跌 且 大盘同步跌 且 RS 守在领先线上方
  主动退潮(要逃) = 板块跌 且 (大盘没事 或 RS跌破1.0且持续走低)
  ```

**验证（2026）**：
- 半导体三月：跟大盘一起跌（300回调6%），但 **RS 始终守在 1.0 上方**（1.04~1.06）→ 被动承压 → 随后 RS 飙到 1.84，真主线。
- 商业航天一月：大盘几乎平，它自己跌12%，**RS 从 1.29 崩到跌破 1.0 再没回去**（6月0.84，连大盘都跑输）→ 主动退潮 → 真死亡。

**严格 RS 过滤（RS 接近自身60日高）回测**：商业航天**全程未触发买入（0%）**，完美规避退潮死板块；但代价是赢家入场推迟（半导体97%→83%）。
> 所以 **RS 用于"板块准入"（可以慢、要确定性），不用于"入场时点"（会拖累时机）**。

⚠️ **RS 的滞后特性**：RS 比值本身零延迟（和价格同步见顶），但"RS 确认退潮"的信号严重滞后——以"RS<1.0"判退潮，商业航天要等到 03-17 才触发，滞后顶部近 2 个月。**所以 RS 绝不能当出场闹钟，只能当事后"要不要再进来"的准入裁判。**

### 4.2 板块宽度（成分股创新高家数）——底部重启的领先信号

- ETF 自身量价对顶底预测**几乎全错**（缩量新高在主升浪里是惯例=看涨，而非顶背离）；真正有效的信号在**穿透到成分股层**。
- **宽度推力领先底部**：半导体四月底，宽度从 0 跳到 13%~16% 时价格还趴在底部，真正拉升在两三周后——**宽度推力比"价格新高回补"更早给出绿灯**。
- **判真假退潮**：半导体退潮后宽度能推回 53% → 价格创新高 → 真重启；商业航天宽度封顶 23% 起不来 → 真死亡。

⚠️ **不要用宽度的"绝对占比"做阈值**：决策当下商业航天宽度(21%)甚至比半导体(16%)还高，绝对值分不出真假。**决定性的是宽度扩张有没有被"价格创新高"确认**——半导体价格实打实破前高再创新高，商业航天价格做更低的高点（1.95→1.70→1.67→1.51）。

## 5. 预警层：跌停潮 / 背离（报警收紧，绝不自动清仓）

### 5.1 跌停潮——最锋利、最快的顶部转向探测器

- **验证（商业航天2026-01）**：01-12 涨停潮(7家涨停) → **01-13 隔天跌停潮(13家单日跌>7%，占29%)** → 01-15、01-26 反复跌停潮。"涨停潮→跌停潮翻转"是教科书级派发顶。
- **比宽度更快**：宽度 01-12 还在最高点 59%，到 01-19 才崩；跌停潮 01-13 当天就报警，**快了整整一周**，补上了宽度"顶部只能同步不能领先"的短板。

### 5.2 真退潮跌停潮的三个语境判据（裸数字不是裁判）

半导体 02-02 也有 14家跌>7%，比商业航天还多，但它没死。区别在语境：

```
真退潮 = 跌停潮 且 ① 前面有涨停潮/已大涨(高位)
                 且 ② 板块比大盘明显弱(不是普跌错杀)
                 且 ③ 后续反复出现(不是一次性)
```
满足 = 高位派发，立即减仓/清仓；低位/跟大盘/一次性 = 错杀，反而可能是机会。

### 5.3 跌停潮为什么只能当预警、不能自动清仓（回测证据）

把跌停潮当自动清仓，半导体 **+97%→+46%**：因为强趋势天生伴随剧烈洗盘，跌停潮在上涨途中频繁误报（半导体5月那些跌停潮发生在 RS 从1.28飙到1.58的过程中，是洗盘不是顶）。

> **单日跌停潮在实时那一刻分不清"洗盘"还是"派发"——只能报警，不能拍板。要透过 RS 镜片解读：RS 强=洗盘无视，RS 弱=确认退潮。**

### 5.4 预警层的其他燃料

| 信号 | 数据现状 |
|---|---|
| 顶背离 / 头肩顶 | 形态类，难机械化，交 AI + 人工 |
| 资金背离（价涨资金撤） | 板块主力净流入历史可取 |
| 高位滞涨 / 触BOLL上轨回落 | K线+BOLL 已有 |

**预警层动作**：①止损上移到最近结构位 ②战术仓减半 ③给 AI 研判打"派发预警"标记。**不直接清仓**。

## 6. 回补层：治踏空

```
买回 = 收盘创20日新高 且 站上MA60 且 大盘市况OK (+ 板块宽度同步抬升/推力)
```

- 三月退潮卖掉的半导体，四月新高自动买回；商业航天新高不来就一直空着。
- "三月退潮卖出没问题，四月新高买回" —— 卖出零纠结，踏空靠"新高回补"解决。
- 宽度推力可让回补**早于价格新高两三周**，但需配合止损管理低质量回补。

## 9. 已知局限（诚实记录）

1. **多腿趋势捕获打折（结构性，修不掉）**：电池56%、新能源21%。想多吃就得扛深调不出场（放弃回撤保护），鱼与熊掌不可兼得，ST(10,3) 已是最优平衡。
2. **Supertrend 参数敏感**：(10,3) 是甜点，(14,3) 会让半导体腰斩——实盘前不要乱调参数。
3. **样本仍有限**：2025/2026 两年、A股特定行情。换大熊市或风格剧变可能失效，需持续跟踪。
4. **RS / 跌停潮 / 宽度依赖成分股重建**：板块资金流直连东财易被反爬，需走 aktools 或汇总成分股，注意取数稳定性。
5. **ETF 日度份额取不到**：理论上最干净的资金信号，但现有数据栈无法获得日度历史。

## 10. 一页速查

```
板块准入 → RS 相对大盘仍创新高/守领先？  否则=退潮死板块，不碰
环境过滤 → 板块跌时大盘也跌(被动可扛) 还是 独跌(主动要逃)？
进场     → 日线 MACD 金叉（不加 ADX/放量过滤）
持有预警 → 跌停潮/涨停-跌停翻转 + 背离 → 收紧止损、减半仓（透过RS解读，不自动清仓）
清仓     → Supertrend(10,3) 翻空（不加深死叉）
回补     → 价格创20日新高 + 站上MA60 + 大盘OK (+宽度推力可提前)
真假退潮 → 看 RS 能否回到领先 / 宽度能否被价格新高确认
```

**记住：策略是趋势收割机，胜负手在"选对正在走趋势的板块"。出场已打磨到头，下一步重心在选股/准入层。**
